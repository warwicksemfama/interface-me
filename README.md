# 1. Descrição do Projeto

O projeto consiste no desenvolvimento de uma interface de login interativa e responsiva, voltada para simular o acesso a um painel de controle de componentes de hardware denominado WolfTech Components.

A aplicação foi desenvolvida utilizando tecnologias modernas de front-end, com foco em usabilidade, experiência do usuário e design visual atrativo. O sistema permite autenticação simulada, visualização de dados fictícios após login e interação com elementos visuais como animações, ícones e notificações.

## 2. Análise das Metas de Usabilidade

As metas de usabilidade definidas para o sistema incluem:

- Facilidade de uso

    - A interface foi projetada para ser intuitiva, com campos claramente identificados, ícones representativos e feedback visual imediato.

- Eficiência

     - O usuário consegue realizar o login em poucos passos, com validação automática dos campos e mensagens claras de erro ou sucesso.

- Aprendizado rápido

     - O design segue padrões comuns de interfaces de autenticação, reduzindo a curva de aprendizado.

- Prevenção de erros

     - O sistema valida campos vazios e credenciais incorretas, exibindo mensagens de alerta para evitar falhas de uso.

- Acessibilidade básica

     - Foram utilizados rótulos (labels), placeholders e atributos de acessibilidade (aria-label) para facilitar o uso por leitores de tela.

## 3. Análise das Metas de Experiência do Usuário (UX)

### As metas de experiência escolhidas foram:

- Estética e engajamento

    - A interface utiliza efeitos visuais como blur, glow e animações para criar uma experiência moderna e futurista.

- Satisfação do usuário

     - Feedback visual positivo (toast de sucesso, animações e mensagens) gera sensação de conquista após login bem-sucedido.

- Interatividade

     - Elementos como mostrar/esconder senha, loading animado e dashboard simulam um sistema real, aumentando a imersão.

- Sensação de segurança

     - mensagens como “Conexão segura” e ícones de cadeado reforçam a percepção de confiabilidade.
